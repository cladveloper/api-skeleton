# Api Skeleton con Rethink DB
## Importante
- Se debe crear el archivo "logs" en la raíz del proyecto
``````
nano logs
``````

## Entornos
### Desarrollador
Para linux
``````
npm run dev
``````

Para windows
``````
npm run dev-win
``````

### Producción
Para linux
``````
npm run start
``````

Para windows
``````
npm run start-win
``````
